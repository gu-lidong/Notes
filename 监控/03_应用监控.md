1.Nginx

+ wget http://....
+ tar filename
+ useradd -s /sbin/nologin -M www
+ ./configure --prefix=/usr/local/nginx-1.10.1 --user=www --group=www --with-http_ssl_module --with-http_stub_status_module
  + configure是一个shell脚本，用来生成Makefile
+ make && make install
+ ln -s /usr/local/nginx-1.10.1 /usr/local/nginx
+ netstat -ntlp 查看80端口有没有被占用
+ /usr/local/nginx/sbin/nginx -t 验证配置文件
+ /usr/local/nginx/sbin/nginx
